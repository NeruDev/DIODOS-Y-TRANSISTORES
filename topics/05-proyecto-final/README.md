## Descripción

Este módulo integra los conocimientos de todos los módulos anteriores para diseñar fuentes de alimentación reguladas. El proyecto final incluye:

1. **Fuente con regulador transistorizado:** Diseño discreto usando BJT para regulación de voltaje.
2. **Fuente con regulador CI:** Implementación con circuitos integrados (ej. LM317, 78xx).
