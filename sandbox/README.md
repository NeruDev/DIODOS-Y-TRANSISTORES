# Sandbox

> Zona de trabajo libre. El contenido aquí **NO** se indexa automáticamente.

## Propósito

Esta carpeta contiene:
- Experimentos y borradores
- Material de prueba
- Contenido en desarrollo sin estructura obligatoria

## Reglas

1. **Sin validación de formato** - Cualquier estructura es válida
2. **Sin indexación automática** - No aparece en `WIKI_INDEX.md`
3. **Contenido experimental** - Puede contener errores o estar incompleto
4. **No es fuente de verdad** - Los agentes de IA no deben usar este contenido como referencia confiable

## Nota sobre Notas/

El contenido educativo que estaba en las carpetas `Notas/` de cada módulo está siendo migrado a sus respectivos archivos en `theory/`. Las `Notas/` originales contienen material de clase que debe clasificarse y consolidarse en los temas oficiales del temario.

*Este directorio existe para mantener flexibilidad sin contaminar el sistema principal.*
